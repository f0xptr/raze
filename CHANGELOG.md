- Initial project setup.
- Core compression and decompression logic.
- Command-line interface with `pack` and `unpack` commands.

- Initial project documentation (internal code comments).
- CI/CD workflow (GitHub Actions).
- Issue and Pull Request templates.
- Contribution guidelines, contributors list, and code of conduct.
- EditorConfig for consistent coding styles.
- Formatting and linting script.
